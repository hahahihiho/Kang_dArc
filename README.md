#### Local setup

```sh
# clone and move directocy
git clone https://github.com/hahahihiho/Kang_dArc cd Kang_dArc
# install node_modules
npm install
# run local hardhat node (you can also use ganache-cli)
npx hardhat node
# deploy contracts on the node
npm run deploy
# run server
npm run dev
```
